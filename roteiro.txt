Roteiro trabalho java

Classe: 
A classe em Java é um importante componente da programação orientação orientada a objetos que serve para
organizar e definir as características dos objetos dentro do programa.
Ela vai funcionar como um molde que, a partir de seus atributos, podemos criar vários objetos.
Ela possui 2 componentes: 
Atributos (ou variáveis de instância): Define as características que os objetos criados a partir dessa classe terão.
Métodos: Define o comportamento e as ações que os objetos poderão sofrer ou realizar.
É boa prática de programação iniciar o nome das classes com letra maiúscula.


Objetos: 
Foi falado tanto em objetos e suas características, mas afinal, o que eles são? Os objetos são uma instancia da classe, 
eles sofrerão influência dos atributos e métodos da classe a qual pertencem. Fazendo uma analogia simples, é como
se eles fossem a versão concreta de alguma coisa da vida real na versão do programa. No exemplo da faculdade nós temos o objeto "objCurso", que receberá diferentes valores
de acordo com as características passadas na classe "Curso", como um nome.
Para instanciar um objeto, primeiro colocamos a qual classe ele pertence, depois damos um nome para utilizarmos na classe atual, = new e novamente o nome da classe com ()

Métodos: 
O método vai ter a função de fazer o objeto realizar ou sofrer uma ação. Dentro de um método podemos exibir mensagens sobre
determinado objeto, além de poder modificar algum de seus atributos e também calcular valores, como médias.
A estrutura do método é composta por um tipo de retorno ( falar qual usamos no trabalho), um nome e um parametro, que pode ou não ser obrigatório. Todo o 
código fica entre chaves.
Na classe de teste, para chamar um método, devemos colocar primeiro o nome do objeto, ponto e depois o nome do método. O parametro deve ser passado
nesse momento caso for obrigatório.

Referencia entre classes
A referencia entre classes ocorre quando uma classe precisa interagir com a outra, utilizando seus métodos e atributos.
Isso é importante porque assim podemos reaproveitar o código, não sendo necessário criar inumeras classes para usar as mesmas caracteristicas de um objeto,
além de colaborar para a organização e eficiencia do programa.
Para linkar as classes, nós precisamos Indicar qual classe queremos e dar um nome a um objeto para usar na classe atual.

Métodos get e set
É através dos métodos get e set que podemos definir e acessar valores dados aos objetos. O set serve para estabelecer um valor ao objeto e get serve para podermos 
"pegar" e usar esse valor posteriormente.
Isso faz parte do conceito de encapsulamento e por meio dele, é possível que manutenções dentro do programa sejam mais faceis de serem corrigidas, pois o código fica inteiro
contido em apenas uma classe. Além de proteger os atributos, o acesso e modificacao destes são feitos de forma controlada.
